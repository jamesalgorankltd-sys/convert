# WebP CDN Source Maker — Vercel Ready

## Deploy
1. Upload this folder to GitHub.
2. Import the GitHub repository in Vercel.
3. Deploy with default settings.
4. Open the Vercel URL. The dashboard will load at `/`.

## Important
- Paste the exact page/image link in the single URL box.
- The resolver now tries the exact page image first and avoids old/random image links hidden inside query/hash parameters.
- For Chrome extension use, load this same folder as an unpacked extension.

## Cloudinary
Use your Cloudinary cloud name and unsigned upload preset in the setup panel.
